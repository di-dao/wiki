title: "ADR-004: User Session Authorization"
description: This document discusses the ADR-004 proposal for user session authorization. The objective is to simplify the login experience with device native biometrics, compute graph-based relations for zero-knowledge elements and DIDs to ensure privacy, and enable a decentralized system. The proposed solution focuses on preventing user session tracking without consent and facilitating a direct authentication flow using encrypted and anonymous PassKeys. The document also provides definitions of key terms and outlines the economic impact, implementation details, and current status of the proposal.
