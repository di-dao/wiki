# Get Identity by Alias

This endpoint allows you to retrieve a DID Document by its associated alias.

<Row>
  <Col sticky>

<CodeGroup title="Request" tag="GET" label="/id/alias/:alias">

```bash
curl -G https://core.build/id/alias/tester
```

```js
import ApiClient from '@sonrhq/ts-client'

const client = new ApiClient(token)

await client.did.get(example.snr)
```
