We currently support <Tooltip tip="motor-flutter supports the iOS, Android, and macOS targets">Flutter</Tooltip> with plans to
support Javascript, React Native, and Web in the future. Motor Flutter is a Flutter library designed to help developers utilize
Sonr in their apps. You can find the package at [motor.build](https://motor.build).

<br />

# Community Libraries

Third-party libraries that have either been developed by the community or deprecated by the core team.

| Name                                                  | Platforms  | Description                                                             | Status         |
| ----------------------------------------------------- | ---------- | ----------------------------------------------------------------------- | -------------- |
| [`motor-grpc`](https://github.com/sonr-io/motor-grpc) | gRPC, REST | Flutter library designed to help developers utilize Sonr in their apps. | **DEPRECATED** |
| [`motoroid`](https://github.com/sonr-io/motoroid)     | Android    | Node library designed to help developers utilize Sonr in their apps.    | **DEPRECATED** |
| [`MotorKit`](https://github.com/sonr-io/MotorKit)     | iOS, macOS | iOS library designed to help developers utilize Sonr in their apps.     | **DEPRECATED** |
