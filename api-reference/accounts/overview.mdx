title: "Accounts Overview"
sidebarTitle: "Overview"
---

The Auth module in the Cosmos SDK is a core module that provides basic account management and permissioning functionality. It allows developers to create and manage accounts, set and enforce access control policies, and track the total supply of tokens on the blockchain.

The Auth module is essential for building applications that require fine-grained control over access to resources, such as decentralized exchanges, governance systems, and permissioned networks. It is designed to be easily integrable with other modules in the Cosmos SDK, allowing developers to build complex, decentralized applications with ease.
