---
title: Fetch Spendable Balance
api: "GET /cosmos/bank/v1beta1/spendable_balances"
description: SpendableBalances queries the spendable balance of all coins for a single account.
---

<ParamField path="address" type="string" required>
  address is the address to query balances for.
</ParamField>

<ParamField path="pagination.key" type="string">
  key is a value returned in PageResponse.next_key to begin querying the next
  page most efficiently. Only one of offset or key should be set.
</ParamField>

<ParamField path="pagination.offset" type="string">
  offset is a numeric offset that can be used when key is unavailable. It is
  less efficient than using key. Only one of offset or key should be set.
</ParamField>

<ParamField path="pagination.limit" type="string">
  limit is the total number of results to be returned in the result page. If
  left empty it will default to a value to be set by each app.
</ParamField>

<ParamField path="pagination.count_total" type="boolean">
  count_total is set to true to indicate that the result set should include a
  count of the total number of items available for pagination in UIs.
  count_total is only respected when offset is used. It is ignored when key is
  set.
</ParamField>

<ParamField path="pagination.reverse" type="boolean">
  reverse is set to true if results are to be returned in the descending order.
</ParamField>
