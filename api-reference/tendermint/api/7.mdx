---
api: POST /cosmos/tx/v1beta1/txs
title: "Broadcast Transaction"
description: "Broadcast a transaction to the network"
---

### Parameters

<ParamField body="tx_bytes" type="string" required>
  Raw transaction bytes.
</ParamField>

<ParamField body="mode" type="string" required>
  Mode can be sync, async, or block. sync and async modes return after CheckTx.
  block mode returns after DeliverTx.
</ParamField>
